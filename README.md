# EBAC Módulo 8 – Cenários em Gherkin

Este repositório contém os cenários de teste escritos em **Gherkin** para as histórias de usuário do projeto **EBAC-SHOP**.

## Estrutura
- `configurar_produto.feature` → Configuração de produto (cor, tamanho, quantidade).
- `login.feature` → Login na plataforma.
- `checkout.feature` → Cadastro no checkout.
